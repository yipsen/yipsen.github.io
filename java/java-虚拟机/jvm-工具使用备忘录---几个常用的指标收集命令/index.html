JVM 工具使用备忘录 - 几个常用的指标收集命令

2023-03-01 08:34:41

内存镜像

  • 输出较为简单的概要信息
jmap -histo:live <pid> > ~/jmap_20230310
  • 完全镜像出 JVM 的内存内容信息（文件会比较大）
jmap -dump:live,format=b,file=~/dump_20230310.hprof <pid>

线程堆栈

jstack <pid> > ~/jstack_20230310

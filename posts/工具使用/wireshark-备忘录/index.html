Wireshark 备忘录

2022-03-05 08:34:41

过滤操作

IP过滤

ip.addr == x.x.x.x ip.src == x.x.x.x ip.dst == x.x.x.x

端口过滤

tcp.port == 21 udp.port == 53

协议过滤

http arp udp tcp icmp …

组合过滤

ip.addr == x.x.x.x && udp.port == 53

选择过滤条件

  1. 选择数据包
  2. 右键弹出菜单并选择对话过滤器
  3. 选择对应协议，比如UDP
  4. 此时过滤窗口自动追加过滤条件如：(ip.addr eq 10.96.10.237 and ip.addr eq 7.213.37.49) and (udp.port eq 10004 and udp.port eq 8011)

追踪流

  1. 选择数据包
  2. 右键弹出菜单并选择追踪流

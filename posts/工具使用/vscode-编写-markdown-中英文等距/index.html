安装字体

  1. 解压后复制选择的字体文件：sarasa-mono-sc-regular.ttf
  2. 安装字体：复制到目录C:\Windows\Fonts即可
  3. 检查是否安装成功：看是否有字体等距更纱黑体 SC 常规

VSCode 设置

  1. 按F1键弹出搜索框后输入settings.json，选择User的一项

  2. 编辑配置，由于``的英文过于逆天，感觉不适合编程使用，设置仅对 markdown 有效

    {
 "[markdown]": {
     "editor.defaultFormatter": "yzhang.markdown-all-in-one",
     "editor.fontFamily": "'Sarasa Mono SC'",
 },
 "files.autoSave": "afterDelay",
 "editor.fontLigatures": false
}

  3. 必要的话，可以重启 VSCode 看是否生效

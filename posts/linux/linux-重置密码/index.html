Linux 重置密码

2022-02-06 16:31:43

更新或重置普通用户密码

  1. 输入passwd命令更新密码

    passwd <username>

  2. 输入新密码并重新输入以确认

    New password: <password>
Retype new password: <password>

  3. 成功则显示

    passwd: password updated successfully

  4. 示例

    # 输入命令选择需要重置密码的用户如：reader
passwd reader
# 输入密码: ysyhl9t@
ysyhl9t@
# 重新输入密码: ysyhl9t@
ysyhl9t@

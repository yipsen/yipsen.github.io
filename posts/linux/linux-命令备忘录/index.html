Linux 命令备忘录

2022-02-06 16:31:43

1. 杀掉指定进程名的进程

ps -ef | grep <procedure_name> | grep -v grep | awk `{print $2}` | xargs kill -9

2. 查看指定监听端口被哪个进程占用

netstat -tunlp | grep <port>

3. 更新安装列表

# ubuntu
## 更新安装列表
apt-get update

## 升级软件
apt-get upgrade

## 安装软件
apt-get install <software_name>

## 卸载软件及其配置
apt-get --purge remove <software_name>

## 卸载软件及其依赖的安装包
apt-get autoremove <software_name>

## 罗列已安装软件
dpkg --list

4. 定位 Java 异常日志使用

tail -300f GroupServer.log | grep . | sed 's/at/\n/'

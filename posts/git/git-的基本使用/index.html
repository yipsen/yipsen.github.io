Git 的基本使用

2022-03-05 08:34:41

1. 取消 SSL 证书校验

git config --global http.sslVerify false

2. 提交密钥

# 基本使用
git push https://<username>:<password>@github.com/<username>/<repository>.git --all

# 使用样例
git push https://yipsen:ghp_ir6E94TlMJcW9Hq3GANJ1hfFarrJsX1PDKDV@github.com/yipsen/xxx.git --all

3. 代码库迁移

# 镜像拷贝：含历史提交记录
git clone -bare <old_repository_url>

cd <respository_name>

# 镜像推送：含历史提交记录
git push --mirror <new_repository_url>

cd ..

# 新代码拉取
git clone <new_repository_url>

4. 打Tag

# 新增
git tag -a <tag_name> -m "xxx version"

# 推送新增
git push origin <tag_name>

# 删除
git tag -d <tag_name>

# 推送删除
git push origin --delete <tag_name>

# 提交全部 Tag
git push origin --tags

5. 撤销

已commit的撤销 git reset --soft HEAD~1
